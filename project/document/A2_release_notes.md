## 2. v1.0.0（2026-03-02）

### 概要

Markdown ファイルの表示・編集・保存ができるスタンドアロンデスクトップアプリケーション `mdFileReader` の初回リリース。

### 新機能

| 機能 | 説明 |
|------|------|
| ファイルツリー表示 | 選択フォルダ配下の `.md` ファイルを左ペインにツリー表示 |
| Markdown エディタ | CodeMirror 5.65.16 によるシンタックスハイライト付き |
| リアルタイムプレビュー | marked.js による Markdown → HTML プレビュー |
| 文字コード自動判定 | chardet による UTF-8 / Shift-JIS 等の自動検出 |
| ファイル CRUD | 新規作成・削除（確認ダイアログ）・名前変更（同名確認ダイアログ） |
| 保存 | `Ctrl+S` / 保存ボタン、UTF-8 BOM なしで書き込み |
| セキュリティ | パストラバーサル防止（BaseDir 外アクセスをブロック） |
| スタンドアロン動作 | PyInstaller 生成 `.exe` 単一ファイル、インストール不要 |

### 動作環境

- Windows 10 / Windows 11
- インストール不要（任意フォルダに `.exe` を配置して実行）

### 既知の制約

| # | 制約 | 暫定対応 |
|---|------|----------|
| 1 | `createFolder` スロット未実装 | ツリー右クリック「新規ファイル」でフォルダ内に `.md` ファイルを作成するとフォルダが自動生成される（`.gitkeep` ワークアラウンド） |
| 2 | Linux / macOS 非対応 | 次バージョン以降で検討 |
