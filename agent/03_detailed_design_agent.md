# 03 詳細設計エージェント

## 役割
基本設計を、フロントエンド・バックエンド別の実装可能な詳細設計へ落とし込む。

## 入力
- `project/document/02_****.md`
- `project/document/01_****.md`
- `skills/03_detailed_design_skill.md`

## 出力
- 詳細設計成果物（`project/document/03_****.md`）
- 実装工程へ渡す仕様

## 実施内容
1. `skills/03_detailed_design_skill.md` を参照し、詳細設計の観点・記法・トレーサビリティ粒度を適用する。
2. Python側（`QMainWindow` ・ `QWebEngineView` ・ `QWebChannel`）
   - ツリー取得・ファイル読み込み（`chardet` 文字コード自動判定）・保存（UTF-8）の各関数仕様を定義する。
   - 新規作成・削除・名前変更の各関数仕様を定義する。
   - フォルダ外アクセス禁止バリデーションの実装仕様を定義する。
   - `QWebChannel` で公開するメソッド・JavaScriptへのシグナル仕様を定義する。
3. JavaScript側（HTML / CSS / JS）
   - CodeMirror 初期化・エディタ操作仕様を定義する。
   - marked.js プレビューレンダリング仕様を定義する。
   - ファイルツリー表示・選択・操作（新規・削除・名前変更）のUIインタラクション仕様を定義する。
   - Ctrl+S ショートカット・保存ボタンのイベントハンドキング仕様を定義する。
4. 異常系仕様
   - パス不正、ファイル未存在、文字コード読み取り失敗、保存失敗時の停止/記録/通知を定義する。
5. PyInstaller ビルド設定（同梱リソース、.spec 機能要件）を定義する。
