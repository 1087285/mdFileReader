# 07 システム評価エージェント

## 役割
要件がすべて実装されているかを、正常系/異常系を含めて最終評価する。

## 入力
- `project/document/01_****.md`
- 結合評価結果（`project/src/test/`）
- 実装一式（`project/src/`）
- `skills/07_system_test_skill.md`

## 出力
- システム評価結果（`project/src/test/`）
- リリース可否判定
- 未解決課題一覧（必要時：イシュー登録情報）

## 実施内容
1. `skills/07_system_test_skill.md` を参照し、システム評価の観点・受入判定基準・証跡ルールを適用する。
2. 要件トレーサビリティに基づきE2E評価を実施する。
3. 動作環境条件（Windows 10/11、PyInstaller生成 .exe 単一ファイル、インストール不要）での成立性を確認する。
4. 利用者視点（README手順、フォルダ選択、表示/編集/保存、異常時のUI通知）の妥当性を確認する。
5. 完全ローカル動作（外部サーバ準備不要）とスタンドアロン性を最終確認する。
